# EV–V2G–PV Digital Twin App

## Local run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deployment

Push these files to GitHub, then deploy on Streamlit Community Cloud.
The platform will generate a public link.

## Notes

- This package is a deployable digital twin web app aligned with the structure of the EV/V2G/PV study.
- It includes a built-in default half-hour baseline profile and also supports an optional CSV upload.
